# Fairness Monitor Lite

Lightweight fairness monitoring for ML model predictions.

## Getting started

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m fairness_monitor_lite
```

## Project structure

```
fairness_monitor_lite/        # source package
tests/      # tests
requirements.txt
```
